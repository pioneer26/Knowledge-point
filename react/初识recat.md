## react

- React 是Facebook推出的一个用于构建用户界面的 Javascript 库
- 核心专注于试图，目的实现组件化开发

#### 组件化
概念
```
将复杂的页面分割成若干个独立组件，
每个组件包含自己的逻辑和样式，
再将这些组件组合成一个复杂页面。
```

特点：

```
1、可组合
//一个组件可以和其他组件一起使用 或 直接嵌套在另一组件内部

2、可重用
//每个组件都有独立的功能，可以被使用多个场景

3、可维护
//每个小组件仅包含自身逻辑，容易理解和维护
```
组件化优势：

```
即减少了逻辑复杂度又实现了代码重用
```
